# ☁️ Cloud Resume Challenge

A professional cloud-based resume website built using AWS and DevOps tools. This project demonstrates my hands-on experience with cloud infrastructure, static website hosting, content delivery, and Infrastructure as Code.

---

# 📌 Project Overview

This project is part of the Cloud Resume Challenge, where I built and deployed my personal portfolio website using AWS services and DevOps best practices.

The website showcases my profile, technical skills, projects, work experience, and downloadable resume.
